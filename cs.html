CSS Tutorial

Welcome to the CSS tutorial. Here you will learn the basics of CSS and how to use it to style web pages.

Introduction to CSS

CSS stands for Cascading Style Sheets. It is used to style and layout web pages, such as changing the font, color, size, and spacing of your content, or adding animations and other decorative features.

CSS Syntax

CSS is a rule-based language â€” you define rules specifying groups of styles that should be applied to particular elements or groups of elements on your web page.

selector {
    property: value;
}

Example of a CSS Rule

p {
    color: red;
    font-size: 16px;
}

CSS Selectors

Selectors are used to "find" (or select) the HTML elements you want to style.

Example of CSS Selectors

/* Selects all <p> elements */
p {
    color: blue;
}

/* Selects the element with id="example" */
#example {
    background-color: yellow;
}

/* Selects all elements with class="example" */
.example {
    font-weight: bold;
}

CSS Colors

Colors are specified using predefined color names, or RGB, HEX, HSL, RGBA, HSLA values.

body {
    background-color: lightblue;
}

h1 {
    color: navy;
}

p {
    color: #333;
}

CSS Box Model

The CSS box model is a fundamental concept for designing layouts. It consists of margins, borders, padding, and the actual content.

div {
    width: 300px;
    border: 15px solid green;
    padding: 50px;
    margin: 20px;
}

CSS Flexbox

Flexbox is a layout model that allows you to design a complex layout structure more efficiently and predictably than traditional models.

.container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}

CSS Grid

The CSS Grid Layout provides a two-dimensional grid-based layout system, allowing you to manage the layout of complex web applications easily.

.container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
}

.item {
    background-color: lightblue;
    padding: 20px;
    text-align: center;
}

CSS Media Queries

Media queries are a feature of CSS that allows content rendering to adapt to different conditions such as screen size and orientation.

@media (max-width: 600px) {
    body {
        background-color: lightpink;
    }
}

CSS Animations

CSS animations allow you to animate transitions between different CSS styles smoothly.

@keyframes example {
    from {background-color: red;}
    to {background-color: yellow;}
}

.box {
    width: 100px;
    height: 100px;
    background-color: red;
    animation: example 4s infinite;
}

CSS Pseudo-classes

Pseudo-classes are used to define the special state of an element.

a:hover {
    color: green;
}

CSS Pseudo-elements

Pseudo-elements are used to style specific parts of an element.

p::first-line {
    font-weight: bold;
}

CSS Transitions

CSS transitions allow you to change property values smoothly (over a given duration).

.box {
    width: 100px;
    height: 100px;
    background-color: blue;
    transition: background-color 0.5s ease;
}

.box:hover {
    background-color: green;
}

CSS Variables

CSS variables (also known as custom properties) allow you to store values that can be reused throughout your CSS.

:root {
    --main-bg-color: coral;
}

body {
    background-color: var(--main-bg-color);
}

CSS Functions

CSS functions allow for complex values and calculations to be used. Common functions include calc(), var(), and rgb().

div {
    width: calc(100% - 20px);
    background-color: rgba(255, 0, 0, 0.5);
}

Advanced CSS Selectors

Conclusion

This is a basic introduction to CSS, including advanced topics. With this knowledge, you can start styling web pages. As you progress, you'll be able to create complex and beautiful designs.
